# Oblivion — User Guide

Oblivion is a **private cleanup agent** for personal information removal — people-search listings, breach checks, and search suppression. You describe the exposure; it drafts removals and **stops before anything is sent** until you approve the exact disclosure.

**Building an app?** See the [Partner API](/docs/developers/partner-api) to embed the same workflow without your servers holding user PII.

Use the **agent panel** on the right (bottom on mobile). Tap **Continue** in the guide bar when prompted.

**Choosing a template?** See [Templates](/docs/user-guide/templates) for what each preset does, which data it needs, and the full step-by-step process.

---

## Start

1. **Start** → enter your name, pick a template, tap **Start cleanup**.
2. Or type one line in the agent panel and send.
3. **Done:** dashboard open, agent running your route.

---

## Review

1. On **Overview**, confirm which listings are yours — **Confirm** or **Not me**.
2. Paste URLs or **Search again** if needed.
3. Tap **Continue** until approvals are needed.

---

## Approve

1. Open **Approvals** (or **Continue** when the guide says so).
2. Read destination, data categories, and purpose on the card.
3. **Approve** only if it matches what you intend. Nothing sends without this.

---

## Controls

| Button | Does |
|--------|------|
| **Continue** | Next safe step for where you are |
| Agent input | Short commands: `run`, `status`, `explain` |

Use the **sidebar** for Overview, Approvals, and Settings.

---

## Wallet (optional)

Not required for cleanup. Connect from the bottom of the sidebar when you want Smart Account / payment demos.

---

## Stuck?

| Issue | Fix |
|-------|-----|
| No dashboard | Finish **Start cleanup** |
| Blocked | Check **Approvals** |
| Wrong case | **Cases** in the sidebar → new or switch |
